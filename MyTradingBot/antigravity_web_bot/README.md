# Antigravity Live Dictation Web App ⚡

Streamlit web application for real-time NIFTY option chain monitoring using Dhan API.

## Features
- Real-time Smart Money dictation
- Support and Resistance wall detection
- Entry zones, targets, and stop-loss generation
- Visual Option Chain analysis

## Installation
1. Ensure you have Python 3.9+ installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the App
Run the app using Streamlit:
```bash
cd antigravity_web_bot
streamlit run app.py
```

## Usage
- Open the application in your browser.
- Enter your Dhan Client ID and Access Token in the sidebar.
- Enable Auto-Refresh to maintain live metrics.

*Note: Your credentials are used for the session only and are never saved or sent to any third party.*
